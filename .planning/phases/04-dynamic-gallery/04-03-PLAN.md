**Data fetching (useEffect + useCallback):**
```typescript
const fetchItems = useCallback(async () => {
  setLoading(true)
  const supabase = createClient()
  let query = supabase
    .from('gallery_items')
    .select('*')
    .order('display_order', { ascending: true })

  if (search) {
    query = query.ilike('title', `%${search}%`)
  }

  const { data } = await query
  setItems(data ?? [])
  setLoading(false)
}, [search])
```

**CRUD operations:**

1. **Create/Update (handleSubmit):**
   - Validate: title required, description required
   - If imageFile exists: upload to `gallery` bucket with path `gallery/{timestamp}-{filename}`, get path
   - Build payload: `{ title, category, year, description, display_order: parseInt(form.display_order), is_featured: form.is_featured, image_url: uploadedPath ?? editingItem?.image_url ?? null }`
   - If editingItem: `supabase.from('gallery_items').update(payload).eq('id', editingItem.id)`
   - If new: `supabase.from('gallery_items').insert(payload)`
   - Close modal, refetch

2. **Delete (handleDelete):**
   - `supabase.from('gallery_items').delete().eq('id', deleteConfirm)`
   - If item had image_url: `supabase.storage.from('gallery').remove([item.image_url])`
   - Close confirm, refetch

3. **Image upload helper:**
   ```typescript
   async function uploadImage(file: File): Promise<string | null> {
     const supabase = createClient()
     const ext = file.name.split('.').pop()
     const path = `gallery/${Date.now()}.${ext}`
     const { error } = await supabase.storage.from('gallery').upload(path, file)
     if (error) return null
     return path
   }
   ```
