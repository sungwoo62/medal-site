Responsive overrides (annotations only ŌĆö not additional canonical sizes):
- Eyebrow: renders at 12px on mobile (`text-xs`), 14px from `sm:` breakpoint (`sm:text-sm`)
- Heading: renders at 30px on mobile (`text-3xl`), 48px at `md:` (`md:text-5xl`), 60px at `lg:` (`lg:text-6xl`)

Weight rationale:
- 700 bold: all display and label roles (eyebrow, h1, CTA) ŌĆö short strings where maximum visual weight is appropriate
- 400 regular: all running text (subhead, body) ŌĆö long strings where readability at 1.625 line-height matters

## Hero Image Contract

This section is specific to Phase 2 and describes the image placement contract.

### Image File

| Property | Value |
|----------|-------|
| Filename | `hero-bg.jpg` |
| Location | `/public/hero-bg.jpg` |
| Format | JPEG (preferred) or WebP |
| Minimum resolution | 2560 ├Ś 1600px (covers largest retina viewport) |
| Subject | Medals ŌĆö marathon completion medals, award medals, or manufacturing process |
| Mood | Professional, premium, dark-toned (overlay will darken further) |

### Responsive Behavior

| Viewport | CSS Behavior | Expected result |
|----------|-------------|-----------------|
| Mobile (< 640px) | `bg-cover bg-center` | Image crops to center; subject remains visible |
| Tablet (640ŌĆō1024px) | `bg-cover bg-center` | Image scales up; no letterboxing |
| Desktop (> 1024px) | `bg-cover bg-center` | Full-bleed background; minimum height `min-h-screen` |

> The existing code at `page.tsx` line 85 already declares `bg-[url('/hero-bg.jpg')] bg-cover bg-center`. No code change is required for responsive behavior ŌĆö only the image file must be placed in `/public/`.

### Overlay Requirement

The dark gradient overlay (`from-black/60 via-black/50 to-black/70`) is already in place at `page.tsx` line 86. It must remain unchanged. White hero text (`text-white`) over this overlay meets WCAG AA contrast.

### Fallback State

If the image fails to load, the dark gradient still renders via the stacked `div` at line 86. The hero is readable without the background image.
